# Setting Nano Editor

## macOS
```bash
brew install nano nanorc
```

already included in .nanorc\
but if your want to set new .nanorc, run command
```bash
echo 'include "/opt/homebrew/share/nanorc/*.nanorc"' >> ~/.nanorc
```
